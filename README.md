# Wiki AvaloriumOt

Este projeto é um site estilo wiki para o servidor de Tibia AvaloriumOt, pronto para ser publicado no GitHub Pages.

## Estrutura Modular
- `src/components`: Componentes JS reutilizáveis
- `src/pages`: Páginas do site
- `src/styles`: Estilos CSS
- `src/assets`: Imagens e arquivos estáticos

## Como publicar no GitHub Pages
1. Faça o push deste repositório para o GitHub.
2. No repositório, acesse as configurações e ative o GitHub Pages apontando para a branch principal e a pasta raiz (`/`).
3. O site estará disponível no endereço fornecido pelo GitHub Pages.
